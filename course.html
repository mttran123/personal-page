    CST8101

    Computer Essentials

    The course equips students with some computer essentials, such as software, hardware, and learn to configure your laptop environment. Students learn to create backups, install virus protection, and manage files through a basic understanding of the Windows Operating System.

    Introduction to Programming

    The course introduces some foundation concepts of Java programming language, including structures, variables, methods, classes, arrays, coding conventions and documentation.

    CST8215

    Introduction to Database

    The course introduces fundamentals of relational databases design using Entity Relation Diagrams (ERDs), and use Structured Query Language (SQL) to create, modify and query a database.

    MAT8001C

    Technical Mathematics for Computer Science

    Students are introduced computer number systems in addition to Boolean algebra and logic to help solve problems involving computer systems. Students also study the addition and subtraction of vectors using vector components.

    ENL1813T

    Communications

    Students practise writing, speaking, reading, listening, locating and documenting information and using technology to communicate professionally through a combination of lectures, exercises, and independent learning.

    CST8300

    Achieving Success in Changing Environments

    Students apply their critical thinking, decision-making skills to help resolve variety of issues in current society.
